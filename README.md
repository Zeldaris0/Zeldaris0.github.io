# ViShot Downloader Website

Static website for GitHub Pages.

## Files included

- `index.html` — landing page
- `privacy.html` — improved privacy policy draft
- `terms.html` — basic terms page
- `app-ads.txt` — uploaded CAS / ads sellers file
- `CNAME` — custom domain: `vishot-downloader.app`
- `assets/` — CSS, JS, logo, screenshots, Open Graph image
- `robots.txt` and `sitemap.xml`

## Upload to GitHub

1. Open the repository `Zeldaris0.github.io`.
2. Upload all files and folders from this package into the root of the repository.
3. Commit to the `main` branch.
4. Go to **Settings → Pages**.
5. Use **Deploy from branch → main → /root**.
6. Custom domain should be `vishot-downloader.app`.
7. Enable **Enforce HTTPS** once GitHub allows it.
